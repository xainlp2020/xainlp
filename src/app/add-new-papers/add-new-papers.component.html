Local vs. Global

Explanation: A local explanation provides information or jus-tification for the model’s prediction on a specificsingle input, while a global provides that justification by revealing how the model’s predictive process works, independently of any particular input.

Explanation: Self-explaining means the explanation is generated as part of the prediction, whereas\ post-hoc processing means the explanation is generated as a separate step after the prediction is made.
